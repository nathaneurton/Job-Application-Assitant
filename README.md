# Job Application Assistant V1

An end-to-end AI automation pipeline that eliminates manual job research and outreach. Built with n8n, Claude API, and Serper API.

## What It Does

- Ingests a job description
- Extracts role requirements and keywords using Claude
- Runs live company research via Serper API
- Generates a personalized cold outreach email automatically
- Logs all data to a Google Sheets tracker

## Built With

- n8n — workflow automation
- Claude API — structured extraction and email generation
- Serper API — live company research
- Google Sheets — output logging
- JavaScript — data cleaning and transformation

## Why I Built It

Manual job research is slow and inconsistent. This pipeline makes the process repeatable, faster, and more personalized at scale — a practical example of how AI creates operational leverage in a real workflow.
